YouTube Video Player

Copy Youtube Id and Paste in code, To play the video. YouTube Id is prefixed with ' ?v= '

Want to see more videos? Just do the following trick

http://localhost:8080/youtube-access?video=<YouTube Id>
e.g - http://localhost:8080/youtube-access/?video=4WB5y4DAcKE
